Project 1 Simple Movie-Category List

For this project I implemented a Doubly linked list and a circular linked list to make a program that allows the user to store specific names into specific categories. The theme was a movie - category project to give the idea that a movie can be inserted into a category. By using multiple header files I was able to organize my code. I have also implemented Binary search to help find specific nodes in the linked list and delete it.

Project 2 2D TopDown Test Demo Game

This project was created using the Raylib library using c++. It is a test demo game that is used to showcase different elements a game has such as props, enemies, player, and a map. The game also features a basic combat mechanic and a simple health bar. This game was created using multiple files along with the basics of object oriented programming. Along with this I used basic vector math to help the players movement throughout the map.

Project 3 Basic Company Website(Front-End Development) - Tinder Theme
